## 🔧 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/karanhimadri/tour-mate.git
   cd tour-mate/frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
